# Tapp

Tapp is the proof of work for how to build a customer-facing product system on Rails. It merges the cardholder and merchant experiences into one product flow, showing how users can pay, merchants can accept, and Rails can settle the value underneath.

## What Tapp demonstrates

- Consumer onboarding into a Rails-backed wallet flow
- Merchant acceptance of payment
- Settlement visibility on both sides of the flow
- A product experience built on top of the same settlement core

## Why it matters

Tapp is not a separate theoretical example. It is the working reference for how Rails can be used in a real product.
